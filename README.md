# 🛒 Customer Segmentation — RFM + K-Means Clustering

## 📌 Overview
Analyzed 500,000+ retail transactions to segment customers 
into behavioral groups using Python.

## 🛠️ Tools Used
- Python, Pandas, NumPy
- Scikit-learn (K-Means)
- Matplotlib, Seaborn
- VS Code + Jupyter Notebook

## 🔍 Steps Followed
1. Data Cleaning
2. RFM Feature Engineering
3. K-Means Clustering (k=4)
4. Visualization & Insights

## 📈 Key Results
- 4 customer segments identified
- Top 15% customers = 60% of revenue
- Actionable marketing strategies per segment
